1. OBTENÇÃO DOS PARÂMETROS DE CAMPO DE FORÇA

Na realização de simulações de dinâmica molecular com o GROMACS, a construção do arquivo .top é indispensável. Esse arquivo contém informações cruciais sobre as características do sistema, como a divisão de carga atômica, constantes de força e outras informações topológicas. No Solvate Suite, existe um módulo chamado GMDFF que facilita a construção desse arquivo. Esse módulo gerencia o programa Q-force, permitindo obter os parâmetros essenciais para a criação do arquivo de topologia.

$ gmdff co2.com -cpcm water -prc 8

-cpcm :
Indica que os parâmetros para a molécula de CO2 são obtidos sob o efeito do modelo implícito CPCM, no qual o solvente é a água.
-prc :
Indica o número de processadores requisitados para a tarefa, no caso, 8.

Observação: O módulo PACKS já contém vários parâmetros de vários solventes, incluindo para a água. Portanto, não é necessário obter esses parâmetros separadamente, pois já estão presentes no banco de dados do módulo. Teste o comando solvs -field ALL para consultar os solventes presentes no banco de dados do módulo.
